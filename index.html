GNUsound
About GNUsound is a multitrack sound editor for GNOME 1 and 2.
Download The current version is 0.7 (older versions), which was released 14 April 2005. Debian packages are available courtesy of Duck (look in the Repository).
Documentation Read the documentation for installation instructions, usage guide, and FAQs.
